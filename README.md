## Ledvance NFC

### Localazy多语言平台
> Localazy平台地址：[https://localazy.com](https://localazy.com/my/dashboard)   
> Localazy下载脚本以及配置存放在项目根目录的**localazy**文件夹

#### 更新多语言方式   
* 方式一：直接在Android Studio右侧的gradle任务列表找到downloadLocalazyString直接点击执行   

```
FieldAssistant=>Tasks=>localazy=>downloadLocalazyString
```

* 方式二：在项目根目录下执行命令   

``` 
./gradlew downloadLocalazyStrings
```
